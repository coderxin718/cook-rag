# 截图说明

请在此目录下添加以下截图文件，然后在根目录 README.md 中取消对应图片的注释：

| 文件名 | 说明 | 建议尺寸 |
| --- | --- | --- |
| `main-interface.png` | 主界面智能问答（搜索 + 流式回答） | 1200×800 |
| `daily-recommendation.png` | 每日推荐展示 | 1200×800 |
| `recipe-detail.png` | 食谱详情弹窗 / 分类浏览 | 1200×800 |

推荐使用 WebP 格式以减小文件体积，使用前修改 README.md 中对应图片的文件扩展名即可。
